## 1. 프로젝트 개요

- **앱 이름**: (미정)
- **플랫폼**: iOS / Android 네이티브 앱
- **목적**: 반려동물의 건강 정보를 날짜별로 기록하고, 과거 내역을 쉽게 조회할 수 있는 건강관리 앱
- **대상 동물**: 모든 반려동물 (강아지, 고양이, 소동물, 조류, 파충류 등)
- **타겟 사용자**: 반려동물 초보 보호자 ~ 전문 보호자 전반

## 2. 핵심 설계 원칙

1. **날짜 기반 기록**: 모든 기록은 날짜에 연결되며, 캘린더/타임라인으로 과거 내역 조회 가능
2. **단순한 UX 우선**: 초보자도 직관적으로 사용할 수 있도록 UI는 최대한 단순하게 설계
3. **다두 지원**: 여러 마리의 반려동물을 등록하고 빠르게 전환 가능
4. **미디어 첨부**: 사진 및 동영상을 각 기록에 첨부 가능
5. **알림 시스템**: 투약, 접종, 병원 예약 등 주요 일정 리마인더 제공
6. **공유 및 내보내기**: 가족 공유, 수의사 공유, PDF 내보내기 지원
7. **컬러 인디케이터**: 날짜별 건강 상태를 색상으로 직관적으로 표시
8. **응급 대응**: 응급 상황 전용 화면 및 주변 병원 빠른 접근 제공
9. **미디어 우선 기록**: 모든 기록 항목에 사진/동영상 첨부 가능하며, 빠른 촬영을 위한 카메라 즉시 실행을 기본으로 한다
10. **미디어 연속성**: 동일 부위/증상의 날짜별 사진을 나란히 비교할 수 있는 타임라인 뷰를 제공한다

> 모든 기록 항목에는 아래 미디어 첨부 스펙이 공통 적용된다.

- 즉시 촬영 (카메라 바로 실행)

### 3-0-3. 부가 기능

- 확대/핀치줌 지원
- 수의사 공유 시 미디어 선택적 포함/제외

### 3-0-4. 저장 정책
- 클라우드 자동 백업
- 로컬 캐시 (오프라인 접근)
- 용량 관리 (무료/유료 저장 용량 차등)

- 약 라벨 OCR 자동 인식

## 4. 기능 목록 (상세)

### 4-1. 반려동물 프로필 관리

#### 기본 정보
- 동물 종류, 이름, 품종, 생년월일, 성별, 중성화 여부 등록
- 털 색상 / 외형 특징 메모 (분실 신고 시 활용)
- 혈액형 기록 (수술/응급 시 활용)
- 입양/분양 경로 기록 (브리더, 보호소, 펫샵 등)
- 입양일 기념일 알림 (기념일 축하 푸시 알림)
- 동물등록번호 저장
- 마이크로칩 번호 저장
- 보험 가입 여부 및 보험사 연결 (보험 관리와 연동)
- 품종별 표준 체중 / 평균 수명 등 기본 정보 제공

#### 미디어
- 프로필 사진 등록
- 프로필 사진 히스토리 (성장 변화 타임라인 자동 구성)
- 전신 사진 정기 등록 (분실 신고 시 최신 외형 제공)
- 특징 부위 사진 (점, 무늬, 흉터 등 식별 마킹용)
- 동영상 클립 (분실 시 움직임/울음소리 공유용)

#### 다두 관리
- 다두 등록 및 프로필 전환
- QR코드 생성 (분실 시 연락처 표시용)

#### 발정 / 번식 관리 ★ NEW
- 발정 주기 기록 및 다음 발정 예측
- 교배 기록
- 임신 여부 기록 및 출산 예정일 계산
- 출산 기록 (새끼 수, 날짜, 특이사항)
- 새끼 개체별 분리 프로필 생성

#### 무지개다리 / 추모 ★ NEW
- 사망 등록 후 기록 아카이브 보존 (삭제 아닌 추모 보관)
- 추모 페이지 생성
- 함께한 날 수 / 추억 통계 자동 생성
- 추모 앨범 PDF 내보내기
- 기일 알림

### 4-2. 날짜별 건강 일지 ⭐ 핵심

- 캘린더 뷰 + 타임라인 뷰 제공
- 날짜 선택 시 해당일 전체 기록 조회
- 기록 유형별 아이콘 표시 (병원, 투약, 배변 등)
- 기간별 검색 및 카테고리 필터링
- 날짜별 컬러 인디케이터 표시
  - 🟢 좋음 / 🟡 보통 / 🔴 나쁨
- 캘린더에서 건강 흐름 한눈에 파악
- 모든 기록에 사진/동영상 첨부 가능

---

### 4-3. 의료 기록 [상위 카테고리]

#### 4-3-1. 병원 방문 기록
- 방문 날짜, 병원명, 담당 수의사 기록
- 진단명 및 진단 소견 입력
- 처방 내역 연동 (약, 주사, 처치)
- 다음 방문 예정일 등록 → 알림 연동
- 병원 즐겨찾기 및 연락처 저장
- 미디어: 진단서, 영수증 사진 첨부 / 증상 사진·동영상 첨부

#### 4-3-2. 처방 & 투약 관리 ⭐ 핵심
- 처방 등록 (약 이름, 용량, 투여 방법, 투여 기간)
- 실제 투약 기록 (투약 여부 체크 + 시간 기록)
- 투약 완료율 통계 (처방 기간 중 몇 % 투약했는지)
- 주사 기록 (종류, 날짜, 접종 기관)
- 투약 알림 / 리마인더
- 미투약 시 경고 알림
- 약 재고 관리 (남은 수량 트래킹)
- 처방전 만료일 알림
- 약 상호작용 경고 메모 (복수 약 복용 시)
- 미디어: 약 봉투/라벨 사진 (OCR 자동 입력 연동 — 로드맵) / 투약 과정 사진 / 주사 부위 사진 (부위 반응/붓기 경과 추적)

#### 4-3-3. 병력 기록
- 과거 질병 이력
- 수술 이력
- 알레르기 정보
- 만성 질환 관리
- 타임라인 형태로 병력 한눈에 보기
- 미디어: 관련 사진/문서 첨부

#### 4-3-4. 예방접종 / 정기관리
- 접종 이력 카드 (종류별 / 날짜별)
- 다음 접종 예정일 자동 계산
- 심장사상충 / 구충 / 벼룩·진드기 주기 관리
- 동물 종류별 기본 접종 스케줄 가이드 (초보자용)
- 미디어: 접종 증명서 사진 첨부

---

### 4-4. 배변 기록 ⭐ 핵심

- 배변 시간 기록
- 유형 선택 (소변 / 대변)
- 상태 선택 (정상 / 묽음 / 딱딱함 / 혈변 등)
- 배변 위치 기록 (실내 / 실외 구분) — 훈련 진행 상황과 연동
- 배변 횟수 이상 감지 (평균 대비 급증/급감 시 알림)
- 고양이 화장실 모래 교체 기록
- 배변 패턴 통계 (주간 / 월간)
- 미디어: 사진 첨부 / 동영상 첨부 / 날짜별 상태 연속 비교 / AI 분석 연동 (로드맵)

---

### 4-5. 증상 / 컨디션 일지

- 증상 태그 선택 (구토, 기침, 무기력, 식욕부진 등)
- 증상 태그 커스텀 추가 (사용자 정의)
- 증상 지속 기간 추적 (며칠째 지속 중인지 자동 계산)

- 컨디션 점수 (1~5점 간단 체크)
- 컬러 상태 표시 (🟢 🟡 🔴)
- 자유 메모 입력
- 미디어: 사진 / 동영상 첨부 (수의사에게 보여주기용)

#### 4-5-1. 피부 / 외형 상태 ★ NEW
- 신체 부위별 상태 기록 (바디맵 UI — 신체 그림에서 부위 선택)
- 상처/발진/탈모 부위 마킹
- 피부 상태 태그 (건조, 발진, 탈모, 상처, 종양 의심 등)
- 미디어: 부위별 상태 사진 (바디맵 연동) / 날짜별 회복 경과 사진 타임라인 (Before/After) / 가려움증·절뚝거림 등 동작 동영상

#### 4-5-2. 수면 / 휴식 패턴 ★ NEW
- 수면 시작/종료 시간 기록
- 수면 장소 기록 (침대, 켄넬, 소파 등)
- 수면 질 메모 (뒤척임, 코골이, 경련 등)
- 주간 수면 패턴 통계
- 미디어: 수면 중 이상 행동 동영상 첨부

### 4-6. 영양 & 신체 관리 [상위 카테고리]

#### 4-6-1. 식이 / 영양 관리
- 끼니별 기록 (아침 / 점심 / 저녁 / 간식)

- 사료 변경 이력 (소화 문제 추적)
- 특별식 / 처방식 여부 기록
- 음수량 별도 기록
- 미디어: 사료 봉투 사진 (바코드 스캔 연동 — 로드맵) / 급여량 사진 / 식사 장면 사진·동영상 (식욕 상태 기록) / 구토·잔반 사진 (소화 이상 기록)

#### 4-6-2. 체중 / 신체 지표
- 체중 기록 및 변화 그래프 (매일 기록 유도)

- 목표 체중 설정 및 달성 진행률 표시
- 체온 기록
- 식욕 / 활동량 메모
- 음수량 기록
- 품종별 표준 체중 범위 비교
- BCS(체형 점수) 기록 — 비만도 체크
- 성장 곡선 (어린 동물용)
- 치아 상태 기록 (스케일링 주기 관리)
- 눈/귀 상태 기록 (정기 체크 항목)
- 발톱 상태 / 발톱 깎기 기록
- 미디어: 체중계 눈금 사진 / 체형 변화 사진 (BCS 연동) / 월별 전신 사진 타임라인

#### 4-6-3. 치과 / 구강 관리 ★ NEW

- 구강 냄새 상태 메모
- 미디어: 치아 상태 사진 기록

- 수의사 방문 시 바로 꺼내 보여주기
- 날짜별 동일 부위 사진 나란히 비교 뷰

### 4-9. 비용 & 보험 [상위 카테고리]

#### 4-9-1. 비용 지출 관리
- 지출 항목 분류
  - 병원비 / 사료 / 미용 / 간식 / 장난감 / 산책용품
  - 유치원 / 위탁 / 보험료 / 기타 (직접 입력)
- 지출처 기록 (어디서 결제했는지)
- 정기 지출 자동 등록 (매월 보험료, 정기 사료 구독 등)
- 예산 설정 및 초과 알림
- 월별 / 연별 항목별 통계 차트
- 캘린더에서 날짜별 지출 확인
- 연간 총 지출 요약 / 반려동물별 연간 지출 리포트
- 다두 보호자용 반려동물별 / 통합 비용 관리
- 미디어: 지출 영수증 사진 첨부

#### 4-9-2. 보험 관리
- 보험 청구 내역 관리
- 보험 증권 정보 저장
- 청구 가능 항목 체크
- 미디어: 보험 증권 사진/PDF 첨부

### 4-10. 가족 공유 & 권한 관리

- 보호자별 권한 설정 (읽기 / 쓰기)
- 기록 작성자 표시 ("엄마가 투약함")
- 돌봄 담당자 지정 ("오늘 저녁 투약은 아빠 담당")
- 공유 일정 캘린더 (가족 전원이 보는 통합 일정)
- 수의사 전용 뷰 (민감 정보 제외한 의료 요약만 공유)
- 수의사 공유 링크 생성
- 펫시터 임시 공유 (기간 한정 접근 권한)

- 24시간 응급 동물병원 별도 표시
- 단골 병원 빠른 전화 연결
- 미디어
  - 응급 증상 즉시 촬영 (병원 이동 중 수의사에게 미리 전송)
  - 응급 상황 동영상 녹화 (발작, 경련 등 → 진단 보조)
  - 응급 기록 자동 타임스탬프 (증상 시작 시각 자동 기록)
