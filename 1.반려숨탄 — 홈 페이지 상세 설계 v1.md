## 2. 전체 레이아웃 구조

```
┌─────────────────────────────┐  ← 상태바 (44dp, primary-50 배경)
├─────────────────────────────┤  ← 헤더 (56dp, primary-50 배경)
│  [홈]    [🐶 콩이 ▾]   [☰]  │     좌: 페이지 제목 / 중앙: 반려동물 칩 / 우: 햄버거
├─────────────────────────────┤
│  ▼ 스크롤 콘텐츠 영역          │  ← cream-bg 배경, 좌우 패딩 space-4(16dp)
│                             │     하단 패딩 100dp (FAB + 탭바 공간 확보)
│  [ 인사 섹션 ]                │
│  [ 오늘의 상태 3칸 ]           │
│  [ 오늘의 기록 카드 목록 ]       │  ← 없으면 Empty State
│  [ 체중 추세 그래프 카드 ]       │
│  [ 배변 주간 통계 카드 ]         │
│                             │
├─────────────────────────────┤
│  [FAB] 우하단 고정 (bottom 92dp)│  ← primary-400, 56×56dp
├─────────────────────────────┤  ← 하단 탭바 (80dp, white 배경)
│  🏠홈  📅일지  ➕기록  🔔알림  🐾 │
└─────────────────────────────┘
```

## 3. 섹션별 상세 명세

### 3-1. 인사 섹션 (Greeting Section)

| 요소 | 스펙 |
|------|------|
| **날짜 텍스트** | `type-caption` · `gray-500` · "YYYY년 MM월 DD일 요일" 형식 |
| **날씨 칩** | 배경 `primary-50` · 보더 `primary-100` · `radius-full` · `type-caption` · `primary-600` |
| **메인 인사 문구** | `type-heading-2` (20sp · W700) · `gray-900` · 반려동물 이름 부분은 `primary-600` |
| **인사 문구 형식** | "안녕하세요! {이름}는 오늘도 건강한가요? 🐾" |
| **마지막 기록 시각** | `type-caption` · `gray-500` · "마지막 기록 · {시간}" |
| **하단 여백** | `space-5` (20dp) |

> **초보자 배려**: 기록이 전혀 없는 첫 사용 시 "처음이시군요! 콩이의 첫 기록을 남겨볼까요? 🐾" 문구 표시

### 3-2. 오늘의 상태 3칸 (Today Summary Chips)

3칸 그리드로 구성. 각 칩은 동일한 너비 (1:1:1 비율).

#### 공통 스펙

| 속성 | 값 |
|------|-----|
| 배경 | `white` |
| 보더 | `gray-200` · 1dp |
| 보더 레디우스 | `radius-lg` (16dp) |
| 패딩 | 세로 `space-3`(12dp) · 가로 `space-2`(8dp) |
| 상단 컬러 바 | 높이 3dp · 상태에 따라 컬러 변경 (아래 표 참고) |
| 정렬 | 세로 중앙 정렬, 아이콘 → 수치 → 레이블 순 |

#### 상단 컬러 바 상태 규칙

| 상태 | 컬러 토큰 | 조건 |
|------|-----------|------|
| 좋음 | `success-400` (#4ADE80) | 컨디션 4~5점 |
| 보통 | `warning-400` (#FACC15) | 컨디션 3점 |
| 나쁨 | `danger-400` (#F87171) | 컨디션 1~2점 |
| 기록 없음 | `gray-300` | 오늘 기록 없음 |

#### 3칸 구성

| 칸 | 아이콘 | 수치 | 레이블 | 수치 포맷 |
|----|--------|------|--------|-----------|
| 1번 | 🟢/🟡/🔴 (컨디션 점수 기반) | N점 | 컨디션 | 1~5점 |
| 2번 | 💩 | N회 | 배변 | 오늘 총 횟수 |
| 3번 | ⚖️ | N.Nkg | 체중 | 소수점 1자리 |

#### 텍스트 스펙

| 요소 | 토큰 | 컬러 |
|------|------|------|
| 아이콘 | 22sp | — |
| 수치 | `type-body-lg` (16sp · W700) | `gray-900` |
| 레이블 | `type-caption` (12sp · W500) | `gray-400` |

## 9. 구현 주의사항

1. **스크롤 영역 하단 패딩**: FAB(56dp) + 탭바(80dp) + 여유(~16dp) = 최소 **100dp** 확보
2. **FAB z-index**: 콘텐츠보다 위, 모달보다 아래 (z-index: 50)
3. **모달 z-index**: FAB보다 위 (z-index: 200)
4. **그래프 SVG**: viewBox 기준으로 반응형 처리 (`width:100%`)
5. **스크롤바 숨김**: `-webkit-scrollbar { display: none }` 적용
6. **카테고리 컬러 바**: 기록 카드 좌측 4dp 고정 너비 바 (보더 레디우스 없음)
7. **오늘 날짜 강조**: X축 레이블 및 바 그래프 오늘 항목은 `primary-600` + W700
8. **반려동물 이름**: 인사 문구 내 이름은 `primary-600` 컬러 적용
9. **기록 카드 최대 노출**: 홈에서는 최신 4개만 노출, 초과 시 "전체 보기" 활성화
10. **날씨 칩**: 1차 MVP에서는 정적 표시 (실제 날씨 API는 2차 연동 예정)
