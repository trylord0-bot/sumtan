## 2. 전역 규칙 (모든 작업 공통)

1. **기존 위젯 재사용 우선** — `FormShell`, `FormDateTimePicker`, `FormSegmentRow`, `FormInputField`, `FormMemoField`는 수정 없이 그대로 사용
2. **모든 폼의 첫 번째 필드는 `FormDateTimePicker`** — 날짜/시간 공통 필드
3. **모든 폼의 마지막 필드는 `FormMemoField`** — 메모 공통 필드 (MemoForm 제외, 아래 별도 명세 참조)
4. **저장 후 반드시 아래 4개 provider를 invalidate**
   ```
   ref.invalidate(todayRecordsProvider);
   ref.invalidate(recentRecordsProvider);
   ref.invalidate(selectedDateRecordsProvider);
   ref.invalidate(monthRecordsProvider);
   ```
5. **저장 성공 시 `Navigator.pop(context, true)` 호출**
6. **비용(cost) 필드는 반드시 `double`로 파싱** — `double.tryParse()` 사용, null이면 저장하지 않음
7. **dataJson의 빈 문자열 필드는 null로 저장** — `text.isEmpty ? null : text` 패턴 사용

## 3. form_widgets.dart 수정 지시

### 3-1. 신규 위젯 추가: `FormTagSelector`

**목적:** 미리 정의된 옵션 목록에서 항목을 선택하여 뱃지 형태로 추가하고, 개별 삭제할 수 있는 다중 선택 위젯

**동작 명세:**

| 단계 | 동작 |
|------|------|
| 1 | `[+ 추가]` 버튼 클릭 |
| 2 | `options` 리스트를 드롭다운 또는 인라인 리스트로 표시 |
| 3 | 항목 선택 시 `selected` 리스트에 추가 → 뱃지로 렌더링 |
| 4 | 뱃지 우측 `✕` 아이콘 탭 시 해당 항목 삭제 |
| 5 | 이미 선택된 항목은 리스트에서 비활성화(회색) 처리 |

**위젯 시그니처:**
```dart
class FormTagSelector extends StatefulWidget {
  final String label;
  final List<String> options;   // 선택 가능한 전체 옵션 목록
  final List<String> selected;  // 현재 선택된 항목 목록
  final ValueChanged<List<String>> onChanged;

  const FormTagSelector({
    super.key,
    required this.label,
    required this.options,
    required this.selected,
    required this.onChanged,
  });
}
```

**UI 스펙:**

| 요소 | 스펙 |
|------|------|
| 레이블 | `fontSize: 13, fontWeight: w600, color: AppColors.gray700` |
| `[+ 추가]` 버튼 | `AppColors.primary50` 배경, `AppColors.primary600` 텍스트, `radius: AppRadius.radiusFull` |
| 옵션 리스트 | 버튼 아래 인라인 표시, `AppColors.gray100` 배경 컨테이너, `radius: AppRadius.radiusMd` |
| 옵션 항목 | `ListTile` 또는 `InkWell` — 선택된 항목은 `AppColors.gray300` 배경 + 비활성 텍스트 |
| 선택된 뱃지 | `AppColors.primary100` 배경, `AppColors.primary900` 텍스트, `AppColors.primary400` 테두리, `radius: AppRadius.radiusFull` |
| 삭제 아이콘 | `Icons.close`, size: 14, `AppColors.primary600` |
| 뱃지 영역 | `Wrap`, spacing: 8, runSpacing: 8 |

## 8. 주의사항

1. **기존 파일 수정 금지** — `form_widgets.dart`에 `FormTagSelector` 추가 외 기존 위젯 코드는 수정하지 마세요.

5. **nullable 다음 예약일** — `_nextVisit`, `_nextDate`는 `DateTime?`으로 선언, dataJson 저장 시 null이면 해당 키 제외
6. **SnackBar 스타일 통일** — `behavior: SnackBarBehavior.floating`, `backgroundColor: AppColors.gray700`
