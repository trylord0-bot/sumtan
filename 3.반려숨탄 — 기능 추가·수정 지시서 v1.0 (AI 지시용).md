### [신규 화면] SettingsScreen

**파일 생성**: `lib/features/settings/presentation/settings_screen.dart`
**라우트 추가**: `/settings`

```
SettingsScreen
├── SubAppBar (← 뒤로가기 + "설정")
└── ListView
    ├── 섹션: 알림 설정
    │   ├── 푸시 알림 허용 [Toggle]
    │   └── 알림 기본 시간 설정 [TimePicker Row]
    ├── 섹션: 데이터 관리
    │   ├── 데이터 내보내기 (ZIP)
    │   └── 데이터 가져오기
    ├── 섹션: 앱 정보
    │   ├── 앱 버전 (텍스트만, 탭 불가)
    │   ├── 개인정보 처리방침 (외부 링크)
    │   └── 오픈소스 라이선스 (LicensePage)
    └── 섹션: 기타
        └── 앱 초기화 (red-600 텍스트)
```

**푸시 알림 Toggle**
- ON: 신규 알림 스케줄 등록 허용
- OFF: 신규 스케줄 중단 + 기존 등록 알림 전체 취소
- 저장: `SharedPreferences` key `push_notification_enabled`

**알림 기본 시간**
- 탭 → `TimePickerDialog`
- 저장: `SharedPreferences` key `default_alarm_time`
- 알림 추가 폼 시간 필드 기본값으로 pre-fill

**데이터 내보내기 (ZIP)**

내보내기 파일 구조:
```
반려숨탄_백업_YYYYMMDD.zip
├── pets.json
├── alarms.json
├── records.json
└── photos/
    └── {pet_id}_photo.jpg
```

플로우:
1. ListTile 탭 → 로딩 오버레이 표시
2. DB 전체 JSON 직렬화 (pets / alarms / records)
3. 사진 파일 수집 (`photo_url` 경로 기반)
4. `archive` 패키지로 ZIP 압축
5. `share_plus`로 공유 시트 표시
6. 완료 스낵바: `"백업 파일이 생성되었습니다."`

**데이터 가져오기**

플로우:
1. ListTile 탭 → `file_picker`로 `.zip` 선택
2. ZIP 압축 해제
3. JSON 파싱 → DB `INSERT OR REPLACE INTO`
4. 사진 파일 로컬 경로 복원
5. 완료 다이얼로그: `"가져오기가 완료되었습니다. 앱을 재시작해주세요."` → 확인 시 앱 재시작

**앱 초기화**
- 탭 → 경고 다이얼로그
- 내용: `"모든 반려동물 정보와 기록이 영구 삭제됩니다. 이 작업은 되돌릴 수 없습니다."`
- 확인 버튼: `"초기화"` (color: `red-600`)
- 실행: DB 전체 DROP + 재생성 + SharedPreferences 초기화 + 앱 재시작
