# 🏠 반려숨탄 — 홈(Home) 화면 수정 지시서 v1.1

---

## 1. 홈 화면 구성 개요

홈 화면은 아래 5개 섹션을 세로로 나열한 스크롤 화면이다.
상단 앱바와 하단 탭바는 공통 레이아웃에서 제공하므로 홈 화면 내에서 별도 구현하지 않는다.
모든 데이터는 현재 선택된 펫(selectedPet) 기준으로 표시한다.

| 순서 | 섹션명 |
|------|--------|
| ① | 인사 카드 |
| ② | 오늘의 상태 |
| ③ | 오늘의 리마인더 |
| ④ | 오늘의 기록 |
| ⑤ | 이번 주 통계 |

---

## 2. ① 인사 카드 — 우측 펫 아바타

### 분기 규칙

| 조건 | 표시 |
|------|------|
| 사진 등록됨 | 원형 사진 |
| 사진 없음 + 강아지 | 🐶 |
| 사진 없음 + 고양이 | 🐱 |
| 사진 없음 + 기타 | 🐾 |

### 스타일
- 크기: 64×64 원형
- 사진 있을 때: 흰색 테두리 + 그림자
- 사진 없을 때: 연한 초록(primary-200) 원형 배경 위 이모지

---

## 3. ② 오늘의 상태 — 뱃지 구성

### 공통 뱃지 (1~5번, 종 무관)

| 순서 | 아이콘 | 라벨 | 표시 형식 |
|------|--------|------|---------|
| 1 | 💩 | 배변 | N회 |
| 2 | 😊 | 컨디션 | 양호 / 보통 / 나쁨 |
| 3 | 🍚 | 식사 | N끼 |
| 4 | 💊 | 투약 | N/N회 + 진행바 |
| 5 | ⚖️ | 체중 | N.Nkg |

### 6번째 뱃지 — 종별 분기

| 종 | 아이콘 | 라벨 | 표시 형식 |
|----|--------|------|---------|
| 강아지 | 🦮 | 산책 | N회 또는 N분 |
| 고양이 | 🪮 | 빗질 | 완료 / - |
| 기타 | ✂️ | 미용 | 완료 / - |

- 미기록 시: 회색 배경 + 아이콘 흐리게 + 값 "-"
- 투약 뱃지: 하단에 얇은 진행바로 달성률 시각화

---

## 4. ③ 오늘의 리마인더

| 상태 | 조건 | 뱃지 |
|------|------|------|
| 오늘 | 오늘 날짜, 미완료 | 황색 "오늘" |
| 예정 | D-1 ~ D-7 이내 | 초록 "D-N" |
| 지남 | 날짜 경과, 미완료 | 빨강 "지남" |

- 정렬: 지남 → 오늘 → 예정 순
- 항목 없으면 섹션 전체 숨김

---

## 5. ④ 오늘의 기록

- 오늘 기록을 최신순으로 최대 5건 표시
- 초과 시 "전체 보기" 링크 → 일지 화면 이동

---

## 6. ⑤ 이번 주 통계

| 카드 | 유지 여부 |
|------|---------|
| 💩 배변 횟수 바 차트 | ✅ 유지 |
| 🍚 식사 횟수 바 차트 | ✅ 유지 |
| ⚖️ 체중 추이 꺾은선 | ✅ 유지 |
| 💊 투약 달성률 | ❌ **제거** |

- 오늘 요일 바: 초록 강조, 라벨 굵게
- 체중 꺾은선: 최근 4주 주별 마지막 기록 기준, 최근 포인트 강조
