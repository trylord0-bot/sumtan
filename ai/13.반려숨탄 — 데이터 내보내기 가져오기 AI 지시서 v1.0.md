# 📦 반려숨탄 — 데이터 내보내기 / 가져오기 AI 지시서 v1.0

> **대상 프로젝트**: `sumtan_app` (Flutter + Riverpod + sqflite)  
> **작성일**: 2026-05-08  
> **적용 플랫폼**: Android, iOS

---

## 1. 개요

반려숨탄 앱의 모든 데이터(JSON + 이미지 + 영상)를 ZIP 파일로 **내보내기(Export)** 하고,
ZIP 파일을 불러와 앱 데이터를 **복원(Import)** 하는 기능을 구현한다.

---

## 2. 기술 스택 및 패키지

> `pubspec.yaml`에 이미 포함된 패키지 기준으로 추가 설치 없이 구현한다.

| `sqflite` | ^2.3.3 | SQLite DB 접근 |

| `share_plus` | 추가 필요 | 공유 시트 (내보내기용) |

## 5. 내보내기 (Export) 구현

### 5-1. 구현 흐름

```
사용자: "내보내기" 버튼 탭
      ↓
진행률 다이얼로그 표시
      ↓
Isolate (백그라운드 처리)
  ├── DatabaseHelper.instance.database 로 DB 접근
  ├── 전체 테이블 데이터 → data.json 직렬화
  ├── meta.json 생성
  ├── images/ 폴더 파일 복사
  ├── videos/ 폴더 파일 복사
  └── ZIP 압축 생성
      ↓
임시 디렉토리에 ZIP 저장
      ↓
share_plus → OS 공유 시트 열기
(파일 앱 저장 / 구글 드라이브 / 카카오톡 등)
```

### 5-2. export_service.dart

```dart
import 'dart:io';
import 'dart:convert';
import 'package:archive/archive_io.dart';
import 'package:path_provider/path_provider.dart';
import 'package:share_plus/share_plus.dart';
import 'package:flutter/foundation.dart';
import '../../core/database/database_helper.dart';

class ExportService {
  static Future<void> exportData({
    required void Function(double progress, String message) onProgress,
  }) async {
    try {
      onProgress(0.05, 'DB 데이터 읽는 중...');

      // 1. DB에서 전체 데이터 조회
      final db = await DatabaseHelper.instance.database;

      final pets = await db.query('pets');
      final healthRecords = await db.query('health_records');
      final diaryEntries = await db.query('diary_entries');
      final medications = await db.query('medications');
      final vaccinations = await db.query('vaccinations');
      final weights = await db.query('weights');

      onProgress(0.2, 'JSON 변환 중...');

      // 2. data.json 생성
      final dataJson = jsonEncode({
        'schema_version': 1,
        'exported_at': DateTime.now().toIso8601String(),
        'pets': pets,
        'health_records': healthRecords,
        'diary_entries': diaryEntries,
        'medications': medications,
        'vaccinations': vaccinations,
        'weights': weights,
      });

      // 3. meta.json 생성
      final metaJson = jsonEncode({
        'app_name': '반려숨탄',
        'app_version': '1.0.0+1',
        'schema_version': 1,
        'exported_at': DateTime.now().toIso8601String(),
        'platform': Platform.isAndroid ? 'android' : 'ios',
        'total_pets': pets.length,
        'total_records': healthRecords.length,
      });

      onProgress(0.4, '미디어 파일 수집 중...');

      // 4. ZIP 생성
      final tempDir = await getTemporaryDirectory();
      final timestamp = DateTime.now().millisecondsSinceEpoch;
      final zipPath = '${tempDir.path}/반려숨탄_백업_$timestamp.zip';

      final encoder = ZipFileEncoder();
      encoder.create(zipPath);

      // JSON 파일 추가
      final dataFile = File('${tempDir.path}/data.json');
      await dataFile.writeAsString(dataJson);
      encoder.addFile(dataFile, 'data.json');

      final metaFile = File('${tempDir.path}/meta.json');
      await metaFile.writeAsString(metaJson);
      encoder.addFile(metaFile, 'meta.json');

      onProgress(0.6, '이미지 압축 중...');

      // 이미지 폴더 추가
      final appDir = await getApplicationDocumentsDirectory();
      final imagesDir = Directory('${appDir.path}/images');
      if (await imagesDir.exists()) {
        encoder.addDirectory(imagesDir, includeDirName: true);
      }

      onProgress(0.8, '영상 압축 중...');

      // 영상 폴더 추가
      final videosDir = Directory('${appDir.path}/videos');
      if (await videosDir.exists()) {
        encoder.addDirectory(videosDir, includeDirName: true);
      }

      encoder.close();

      onProgress(1.0, '완료!');

      // 5. 공유 시트 열기
      await Share.shareXFiles(
        [XFile(zipPath)],
        subject: '반려숨탄 데이터 백업',
        text: '반려숨탄 앱 데이터 백업 파일입니다.',
      );

      // 임시 파일 정리
      await dataFile.delete();
      await metaFile.delete();

    } catch (e) {
      throw Exception('내보내기 실패: $e');
    }
  }
}
```

### 5-3. export_provider.dart

```dart
import 'package:flutter_riverpod/flutter_riverpod.dart';
import 'export_service.dart';

enum ExportStatus { idle, loading, success, error }

class ExportState {
  final ExportStatus status;
  final double progress;
  final String message;
  final String? errorMessage;

  const ExportState({
    this.status = ExportStatus.idle,
    this.progress = 0.0,
    this.message = '',
    this.errorMessage,
  });

  ExportState copyWith({
    ExportStatus? status,
    double? progress,
    String? message,
    String? errorMessage,
  }) => ExportState(
    status: status ?? this.status,
    progress: progress ?? this.progress,
    message: message ?? this.message,
    errorMessage: errorMessage ?? this.errorMessage,
  );
}

class ExportNotifier extends StateNotifier<ExportState> {
  ExportNotifier() : super(const ExportState());

  Future<void> startExport() async {
    state = state.copyWith(status: ExportStatus.loading, progress: 0.0);
    try {
      await ExportService.exportData(
        onProgress: (progress, message) {
          state = state.copyWith(progress: progress, message: message);
        },
      );
      state = state.copyWith(status: ExportStatus.success, progress: 1.0);
    } catch (e) {
      state = state.copyWith(
        status: ExportStatus.error,
        errorMessage: e.toString(),
      );
    }
  }

  void reset() => state = const ExportState();
}

final exportProvider =
    StateNotifierProvider<ExportNotifier, ExportState>(
  (ref) => ExportNotifier(),
);
```

## 6. 가져오기 (Import) 구현

### 6-1. 구현 흐름

```
사용자: "가져오기" 버튼 탭
      ↓
⚠️ 경고 다이얼로그 표시
"기존 데이터가 모두 삭제됩니다. 계속하시겠습니까?"
      ↓
확인 → file_picker로 ZIP 파일 선택
      ↓
진행률 다이얼로그 표시
      ↓
Isolate (백그라운드 처리)
  ├── ZIP 파일 해제
  ├── meta.json 스키마 버전 검증
  ├── data.json 파싱
  ├── 기존 DB 데이터 전체 삭제
  ├── 새 데이터 DB에 삽입
  ├── images/ → 앱 디렉토리 복원
  └── videos/ → 앱 디렉토리 복원
      ↓
앱 상태 새로고침 (Riverpod invalidate)
      ↓
완료 메시지 표시 → 홈 화면 이동
```

### 6-2. import_service.dart

```dart
import 'dart:io';
import 'dart:convert';
import 'package:archive/archive_io.dart';
import 'package:path_provider/path_provider.dart';
import '../../core/database/database_helper.dart';

class ImportService {
  static const int _supportedSchemaVersion = 1;

  static Future<void> importData({
    required String zipFilePath,
    required void Function(double progress, String message) onProgress,
  }) async {
    try {
      onProgress(0.05, 'ZIP 파일 읽는 중...');

      // 1. ZIP 파일 해제
      final bytes = await File(zipFilePath).readAsBytes();
      final archive = ZipDecoder().decodeBytes(bytes);

      final tempDir = await getTemporaryDirectory();
      final extractPath = '${tempDir.path}/import_temp';
      final extractDir = Directory(extractPath);
      if (await extractDir.exists()) await extractDir.delete(recursive: true);
      await extractDir.create();

      onProgress(0.1, '파일 압축 해제 중...');

      // 2. 파일 압축 해제
      for (final file in archive) {
        final filePath = '$extractPath/${file.name}';
        if (file.isFile) {
          final outFile = File(filePath);
          await outFile.create(recursive: true);
          await outFile.writeAsBytes(file.content as List<int>);
        } else {
          await Directory(filePath).create(recursive: true);
        }
      }

      onProgress(0.2, '데이터 검증 중...');

      // 3. meta.json 검증
      final metaFile = File('$extractPath/meta.json');
      if (!await metaFile.exists()) {
        throw Exception('유효하지 않은 백업 파일입니다. (meta.json 없음)');
      }

      final meta = jsonDecode(await metaFile.readAsString());
      final schemaVersion = meta['schema_version'] as int? ?? 0;

      if (schemaVersion > _supportedSchemaVersion) {
        throw Exception(
          '지원하지 않는 백업 버전입니다. 앱을 최신 버전으로 업데이트해 주세요.\n'
          '(백업 버전: $schemaVersion, 지원 버전: $_supportedSchemaVersion)',
        );
      }

      onProgress(0.3, 'JSON 데이터 파싱 중...');

      // 4. data.json 파싱
      final dataFile = File('$extractPath/data.json');
      if (!await dataFile.exists()) {
        throw Exception('유효하지 않은 백업 파일입니다. (data.json 없음)');
      }

      final data = jsonDecode(await dataFile.readAsString());

      onProgress(0.4, '기존 데이터 초기화 중...');

      // 5. 기존 DB 데이터 전체 삭제
      final db = await DatabaseHelper.instance.database;
      await db.transaction((txn) async {
        await txn.delete('weights');
        await txn.delete('vaccinations');
        await txn.delete('medications');
        await txn.delete('diary_entries');
        await txn.delete('health_records');
        await txn.delete('pets');
      });

      onProgress(0.55, '데이터 복원 중...');

      // 6. 새 데이터 삽입
      await db.transaction((txn) async {
        for (final pet in (data['pets'] as List? ?? [])) {
          await txn.insert('pets', Map<String, dynamic>.from(pet));
        }
        for (final record in (data['health_records'] as List? ?? [])) {
          await txn.insert('health_records', Map<String, dynamic>.from(record));
        }
        for (final entry in (data['diary_entries'] as List? ?? [])) {
          await txn.insert('diary_entries', Map<String, dynamic>.from(entry));
        }
        for (final med in (data['medications'] as List? ?? [])) {
          await txn.insert('medications', Map<String, dynamic>.from(med));
        }
        for (final vac in (data['vaccinations'] as List? ?? [])) {
          await txn.insert('vaccinations', Map<String, dynamic>.from(vac));
        }
        for (final weight in (data['weights'] as List? ?? [])) {
          await txn.insert('weights', Map<String, dynamic>.from(weight));
        }
      });

      onProgress(0.7, '이미지 복원 중...');

      // 7. 미디어 파일 복원
      final appDir = await getApplicationDocumentsDirectory();

      final importedImagesDir = Directory('$extractPath/images');
      if (await importedImagesDir.exists()) {
        final targetImagesDir = Directory('${appDir.path}/images');
        if (await targetImagesDir.exists()) {
          await targetImagesDir.delete(recursive: true);
        }
        await _copyDirectory(importedImagesDir, targetImagesDir);
      }

      onProgress(0.85, '영상 복원 중...');

      final importedVideosDir = Directory('$extractPath/videos');
      if (await importedVideosDir.exists()) {
        final targetVideosDir = Directory('${appDir.path}/videos');
        if (await targetVideosDir.exists()) {
          await targetVideosDir.delete(recursive: true);
        }
        await _copyDirectory(importedVideosDir, targetVideosDir);
      }

      onProgress(0.95, '임시 파일 정리 중...');

      // 8. 임시 파일 정리
      await extractDir.delete(recursive: true);

      onProgress(1.0, '가져오기 완료!');

    } catch (e) {
      throw Exception('가져오기 실패: $e');
    }
  }

  /// 디렉토리 재귀 복사
  static Future<void> _copyDirectory(Directory source, Directory target) async {
    await target.create(recursive: true);
    await for (final entity in source.list(recursive: false)) {
      if (entity is File) {
        await entity.copy('${target.path}/${entity.uri.pathSegments.last}');
      } else if (entity is Directory) {
        final newDir = Directory('${target.path}/${entity.uri.pathSegments.last}');
        await _copyDirectory(entity, newDir);
      }
    }
  }
}
```

## 9. 성능 및 UX 가이드라인

| 항목 | 가이드 |
|------|--------|
| **스레드** | ZIP 생성/해제는 반드시 `compute()` 또는 `Isolate` 사용 |
| **진행률 UI** | `LinearProgressIndicator` + 단계별 메시지 표시 |
| **대용량 영상** | 100MB 이상 파일 포함 시 예상 시간 안내 |
| **오류 처리** | 실패 시 부분 생성된 파일 자동 삭제 |
| **파일명** | `반려숨탄_백업_YYYYMMDD_HHmm.zip` 형식 사용 |
| **가져오기 전 경고** | 기존 데이터 삭제 경고 다이얼로그 필수 |
| **가져오기 후 처리** | Riverpod `ref.invalidate()` 로 전체 상태 초기화 |

## 10. 보안 고려사항

- 민감 데이터(건강 기록 등) 포함 시 ZIP 암호화 옵션 제공 고려
- 가져오기 시 `schema_version` 검증으로 호환성 확인
- 외부 저장소 직접 접근 대신 `share_plus` 공유 시트 사용 권장
