# 🐾 AppBar 반려동물 선택 버튼 정중앙 고정 수정 지시서

## 📌 문제 설명

현재 앱의 `AppBar` 상단 중앙에 위치한 **반려동물 선택 버튼**이, 화면(라우트)마다 좌측에 표시되는 요소(뒤로가기 버튼, 햄버거 메뉴, 화면 타이틀 텍스트 등)의 너비 차이로 인해 **버튼이 좌우로 미세하게 이동하는 문제**가 있습니다.

### 원인
Flutter의 `AppBar`는 `leading` 영역과 `actions` 영역의 **실제 렌더링 너비가 다를 경우**, `centerTitle: true`를 설정해도 `title` 위젯이 진정한 화면 정중앙에 배치되지 않습니다.  
Flutter는 `title`을 `leading`과 `actions` 사이의 **남은 공간의 중앙**에 배치하기 때문입니다.

---

## ✅ 수정 목표

- 반려동물 선택 버튼을 **화면 너비 기준 정확한 정중앙**에 고정
- 어떤 화면에서도 버튼 위치가 흔들리지 않도록 고정
- 기존 `leading`, `actions` 구성은 유지

---

## 🛠️ 수정 방법

### 방법 1 (권장): `AppBar`의 `title`을 `Stack` + `Positioned` 구조로 교체

`AppBar` 자체를 `PreferredSize` + `Stack` 구조로 감싸서, 반려동물 선택 버튼을 **화면 전체 너비 기준 절대 중앙**에 배치합니다.

#### 적용 위치
- 파일: `lib/` 하위에서 공통 AppBar를 정의하는 위젯 파일  
  (예: `lib/widgets/common_app_bar.dart` 또는 각 화면의 `Scaffold(appBar: ...)` 부분)

#### 수정 코드 예시

```dart
// 기존 AppBar를 아래와 같이 교체하세요.

AppBar(
  leading: /* 기존 leading 위젯 유지 */,
  actions: [ /* 기존 actions 유지 */ ],
  // ✅ title을 null 또는 빈 위젯으로 설정
  title: null,
  // ✅ flexibleSpace를 사용해 정중앙에 버튼 고정
  flexibleSpace: SafeArea(
    child: Stack(
      alignment: Alignment.center,
      children: [
        // 반려동물 선택 버튼을 Stack의 center에 배치
        Center(
          child: PetSelectorButton(), // 기존 반려동물 선택 버튼 위젯
        ),
      ],
    ),
  ),
)
```

---

### 방법 2: `titleSpacing` + `toolbarHeight` 조정 없이, `title`에 `LayoutBuilder` 사용

`AppBar`의 `title` 자리에 아래처럼 `SizedBox`로 화면 전체 너비를 강제 지정합니다.

```dart
AppBar(
  leading: /* 기존 leading 유지 */,
  actions: [ /* 기존 actions 유지 */ ],
  centerTitle: true,
  titleSpacing: 0, // ✅ title 영역의 좌우 여백 제거
  title: SizedBox(
    width: double.infinity,
    child: Center(
      child: PetSelectorButton(), // 기존 반려동물 선택 버튼 위젯
    ),
  ),
)
```

> ⚠️ 이 방법은 `leading`/`actions` 너비 차이가 클 경우 여전히 미세하게 밀릴 수 있습니다.  
> **방법 1(flexibleSpace)을 우선 적용하세요.**

---

### 방법 3 (가장 확실): `leading`과 `actions`를 동일 너비로 맞추기

`leading` 영역과 `actions` 영역의 너비를 동일하게 맞추면 Flutter가 자동으로 `title`을 정중앙에 배치합니다.

```dart
AppBar(
  centerTitle: true,
  // ✅ leading을 SizedBox로 감싸 고정 너비 지정
  leadingWidth: 56, // actions 총 너비와 동일하게 맞춤
  leading: /* 기존 leading 위젯 */,
  title: PetSelectorButton(), // 반려동물 선택 버튼
  actions: [
    // ✅ actions 전체를 SizedBox로 감싸 고정 너비 지정
    SizedBox(
      width: 56, // leadingWidth와 동일한 값
      child: /* 기존 actions 위젯 */,
    ),
  ],
)
```

## 📂 수정 대상 파일 탐색 방법

아래 파일들을 순서대로 확인하여 반려동물 선택 버튼이 정의된 위치를 찾으세요:

1. `lib/app/router.dart` — 라우트별 화면 구성 확인
2. `lib/widgets/` 또는 `lib/common/` — 공통 AppBar 위젯 탐색
3. 각 화면 파일의 `Scaffold(appBar: ...)` 부분 탐색
4. 반려동물 선택 버튼 위젯 클래스명 검색 (예: `PetSelector`, `PetButton`, `ActivePetButton` 등)

## 💡 참고 사항

- Flutter의 `AppBar.centerTitle`은 **"leading과 actions 사이 공간의 중앙"** 이지, **"화면 전체의 중앙"** 이 아닙니다.
- `flexibleSpace`는 AppBar 전체 영역(leading/actions 포함)을 기준으로 렌더링되므로, `Center` 위젯을 사용하면 **항상 화면 정중앙**에 고정됩니다.
- `SafeArea`를 `flexibleSpace` 내부에 반드시 포함하여 상태바 영역을 고려하세요.
