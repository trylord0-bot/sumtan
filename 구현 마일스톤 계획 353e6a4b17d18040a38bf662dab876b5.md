## 1. 마일스톤 전략 개요

기능이 방대한 만큼, **3단계 점진적 배포 전략**을 채택한다.
각 마일스톤은 독립적으로 완결성을 가지며, 이전 버전 사용자가 자연스럽게 다음 버전으로 전환될 수 있도록 설계한다.

| 마일스톤 | 타겟 | 핵심 키워드 | 예상 개발 기간 |
| --- | --- | --- | --- |
| **1차 (MVP)** | 초보 보호자 | 기록 습관 형성 | 3~4개월 |
| **2차** | 일반 보호자 | 관리 고도화 | 3~4개월 |
| **3차** | 전문/파워 유저 | 연결 & 확장 | 4~6개월 |

## 2. 1차 마일스톤 개요

- **대상**: 반려동물 초보 보호자
- **컨셉**: "처음 반려동물을 키우는 분도 5분 안에 시작할 수 있는 앱"
- **핵심 목표**: 복잡한 기능은 모두 숨기고, 핵심 기록 습관 형성에만 집중
- **설계 원칙**:
    - 온보딩 3단계 이내 완료 (동물 등록 → 첫 기록 → 알림 설정)
    - 초보자도 직관적으로 사용할 수 있도록 UI는 최대한 단순하게 설계
    - 각 기록 화면에 "이렇게 활용하세요" 가이드 문구 내장
    - 기록이 없을 때 예시 화면(Empty State) 제공
    - 빠른 기록 FAB 버튼으로 홈에서 주요 기록 즉시 접근

## 3. 1차 포함 기능 목록 (상세)

### 3-1. 반려동물 프로필 (기본)

- 동물 종류, 이름, 품종, 생년월일, 성별, 중성화 여부 등록
- 프로필 사진 등록
- 다두 등록 및 전환 (최대 3마리)
- 품종별 표준 체중 / 평균 수명 기본 정보 제공

> **초보자 배려**: 등록 시 "왜 이 정보가 필요한가요?" 툴팁 제공
> 

---

### 3-2. 날짜별 건강 일지 (기본)

- 캘린더 뷰 제공
- 날짜별 기록 목록 조회
- 컬러 인디케이터 표시
    - 🟢 좋음 / 🟡 보통 / 🔴 나쁨
- 기록 유형별 아이콘 표시 (배변, 투약, 컨디션 등)

---

### 3-3. 증상 / 컨디션 일지 (기본)

- 컨디션 점수 (1~5점 간단 체크)
- 증상 태그 선택 (구토, 기침, 무기력, 식욕부진 등 기본 태그 제공)
- 자유 메모 입력
- 사진 첨부 (1장)

> **초보자 배려**: "이런 증상이 보이면 기록하세요" 예시 가이드 제공
> 

---

### 3-4. 배변 기록 (기본)

- 배변 시간 기록
- 유형 선택 (소변 / 대변)
- 상태 선택
    - 정상 / 묽음 / 딱딱함 / 혈변
- 배변 횟수 주간 통계 (간단 그래프)

> **초보자 배려**: 상태 선택 시 각 상태의 의미와 주의사항 설명 팝업 제공
> 

---

### 3-5. 예방접종 / 정기관리 (기본)

- 접종 이력 등록 (종류, 날짜)
- 다음 접종 예정일 자동 계산
- 동물 종류별 기본 접종 스케줄 가이드 제공 (초보자 필수)
- 심장사상충 / 구충 주기 관리

> **초보자 배려**: "우리 아이에게 필요한 접종이 뭔가요?" 가이드 카드 제공
> 

---

### 3-6. 처방 & 투약 관리 (기본)

- 처방 등록 (약 이름, 용량, 투여 방법, 투여 기간)
- 실제 투약 여부 체크
- 투약 알림 / 리마인더

---

### 3-7. 체중 기록 (기본)

- 체중 기록 및 변화 그래프
- 품종별 표준 체중 범위 비교
- 목표 체중 설정

---

### 3-8. 알림 & 리마인더 (기본)

- 투약 시간 알림
- 정기 예방접종 알림
- 심장사상충 / 구충제 주기 알림
- 기념일 알림 (생일, 입양일)

---

### 3-9. 응급 화면 (기본)

- 응급 상황 전용 화면 (원터치 접근)
- 위치 기반 주변 동물병원 찾기
- 24시간 응급 동물병원 별도 표시
- 단골 병원 빠른 전화 연결

## 5. 화면 구조 (네비게이션)

```
🏠 홈 (오늘의 기록 요약 + 컨디션 인디케이터)
│
├── 📅 건강 일지 (캘린더)
│   └── 날짜 선택 → 당일 기록 조회/추가
│
├── ➕ 빠른 기록 (FAB 버튼)
│   ├── 💩 배변 기록
│   ├── 🌡️ 컨디션 기록
│   ├── 💊 투약 체크
│   └── ⚖️ 체중 기록
│
├── 🐾 내 반려동물
│   └── 프로필 / 접종 / 체중 관리
│
├── 🚨 응급 (상단 고정 버튼)
│
└── ⚙️ 설정
    └── 알림 관리
```

## 6. UX 설계 원칙 (1차)

1. **온보딩 3단계**: 동물 등록 → 첫 기록 → 알림 설정
2. **빠른 기록 FAB**: 홈 화면에서 탭 한 번으로 주요 기록 접근
3. **가이드 문구 내장**: 각 기록 화면에 "이렇게 활용하세요" 안내 상시 노출
4. **Empty State 친화적**: 기록이 없을 때 예시 화면 및 첫 기록 유도 CTA 제공
5. **툴팁 제공**: 입력 항목마다 초보자를 위한 설명 툴팁 제공
6. **다크모드 지원**: 기본 제공

## 7. 미디어 첨부 스펙 (1차 적용 범위)

> 1차에서는 미디어 첨부를 **증상/컨디션 기록에 한해 사진 1장**으로 제한한다.
전체 미디어 스펙은 2차 이후 단계적으로 확대 적용한다.
> 

### 7-1. 입력 방식

- 즉시 촬영 (카메라 바로 실행)
- 갤러리에서 선택

### 7-2. 지원 형식

- 사진: JPG / PNG / HEIC

### 7-3. 부가 기능

- 자동 타임스탬프 기록
- 썸네일 미리보기

### 7-4. 저장 정책

- 클라우드 자동 백업
- 로컬 캐시 (오프라인 접근)
