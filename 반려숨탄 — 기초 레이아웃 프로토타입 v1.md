## 6. 탭별 페이지 개요

### 6-1. 홈 (Home)
- 오늘 날짜 인사 문구
- 컨디션 / 배변 / 체중 요약 칩 3개 (가로 배열)
- 오늘의 기록 요약 카드 목록 (`primary-50` 기본 카드)
- 체중 추세 그래프 카드 (추후 구현)

### 6-2. 일지 (Journal)
- 월별 캘린더 뷰 (7열 그리드)
- 날짜별 컬러 인디케이터 점 (🟢🟡🔴)
- 선택 날짜 기록 목록 (기록 카드 — 상단 컬러 바 포함)

### 6-3. 기록 (Record)
- 기록 유형 선택 그리드 (3열)
- 유형: 컨디션, 배변, 투약, 식사, 체중, 병원, 접종, 미용, 메모

### 6-4. 알림 (Alarm)
- 예정된 알림 목록
- 알림별 토글 스위치 (ON: `primary-400` / OFF: `gray-300`)

### 6-5. 프로필 (Profile)
- 반려동물 프로필 히어로 카드
- 설정 메뉴 목록 (프로필 수정, 추가 등록, 내보내기, 알림 설정, 앱 설정 등)

## 7. 인터랙션 규칙

| 인터랙션 | 동작 |
|---------|------|
| 탭 전환 | 헤더 제목 동적 변경 + 해당 페이지 표시 |
| 햄버거 버튼 탭 | 우측 드로어 오픈 + 아이콘 X 전환 |
| 드로어 오버레이 탭 | 드로어 닫힘 + 아이콘 햄버거 복귀 |
| 반려동물 칩 탭 | 반려동물 전환 바텀시트 오픈 (추후 구현) |
| 중앙 기록 탭 탭 | 기록 유형 선택 화면 진입 |
| 기록 카드 탭 | 해당 기록 상세 화면 진입 (추후 구현) |

## 8. 구현 시 주의사항

1. **탭 전환은 라우팅이 아닌 show/hide 방식** — 각 탭 페이지는 DOM에 유지하고 `display` 속성으로 전환
2. **중앙 기록 탭은 탭바 위로 돌출** — `position: relative; top: -16dp` 처리
3. **드로어는 z-index 관리 필수** — overlay: 100, drawer: 101
4. **콘텐츠 영역 하단 패딩** — 탭바(80dp)만큼 `padding-bottom` 확보
5. **헤더와 상태바는 동일 배경(`primary-50`)** — 시각적으로 하나의 영역처럼 보이도록
6. **스크롤바 숨김** — 콘텐츠 영역 `::-webkit-scrollbar { display: none }`
7. **모든 컬러는 토큰명 참조** — HEX 직접 사용 금지 (CSS 변수로 관리)
