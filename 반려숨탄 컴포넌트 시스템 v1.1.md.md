# 🧩 반려숨탄 — 컴포넌트 시스템 명세서 v1.1 (AI 지시용)

## 📌 역할 지시
당신은 모바일 앱 UI/UX 디자인 전문가입니다.
아래 컴포넌트 시스템을 기반으로 요청하는 디자인 작업을 수행해 주세요.
반드시 토큰명으로 참조하고, 수치를 직접 사용하지 마세요.
이 문서는 아래 두 문서와 함께 사용한다.

| 연관 문서 | 내용 |
|-----------|------|
| 컬러 시스템 v1.0 | Primary, Neutral, Semantic, Category 컬러 토큰 |
| 타이포그래피 & 스페이싱 시스템 v1.0 | 폰트, 타입 스케일, 스페이싱, 보더 레디우스 토큰 |

## 🔑 전역 규칙 (모든 컴포넌트 공통)

1. **Filled(배경 있는) 컴포넌트 위 텍스트는 `white`** — primary-900 사용 금지
2. **Outlined / Ghost 컴포넌트 텍스트는 `primary-600` 또는 `gray-500`** — white 사용 금지
3. **Disabled 상태는 항상 배경 `gray-200`, 텍스트 `gray-400`**
4. **모든 버튼 텍스트는 `type-label` (13sp · W500)**
5. **앱 전체 배경은 `cream-bg` (`#FFFDF7`)** 고정
6. **헤더 배경은 `primary-50` (`#ECFDF5`)** 고정
